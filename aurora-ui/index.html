자주 묻는 질문

네. Bootstrap, Tailwind 등 어떤 라이브러리도 사용하지 않고 reset부터 컴포넌트까지 전부 직접 작성했습니다.
localStorage에 저장되며, 처음 방문 시에는 운영체제 테마를 자동으로 감지합니다.
prefers-reduced-motion을 감지해 모든 애니메이션 지속시간을 자동으로 최소화합니다.
막대 차트는 CSS, 선 차트는 inline SVG path, 도넛/게이지는 conic-gradient로 직접 구현했습니다.
