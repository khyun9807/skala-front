# myHoliday 페이지 콘텐츠 — 나의 휴일 일과

> `html/myHoliday.v4.html`을 채울 내용. 사용자가 "알아서 잘 넣어줘"라고 위임 → 아래는 **제안(수정 가능)**. 저장일 2026-07-16.
> 톤 참고: 백엔드 취준생, 외향형(7), 좋아하는 음식 수육·피자·갈비, 목표 금융권 취업·여행. (억지스럽지 않게, 편하게 쉬는 휴일 컨셉.)

## 🌅 느긋한 아침 (제안)
알람 없이 몸이 깨어날 때까지 푹 잔다. 눈 뜨면 침대에서 유튜브·OTT를 잠깐 보다가, 느지막이 일어나 라면이나 배달 브런치로 하루를 연다. 평일의 알람과 출석 체크에서 벗어나는 이 늦잠이 휴일의 가장 큰 사치.

## 🚶 활기찬 오후 (제안)
가볍게 몸을 풀러 헬스장에 가거나 동네를 산책한다. 돌아오는 길엔 카페에 자리를 잡고 코딩 테스트·이력서를 조금 손보거나 사이드 프로젝트를 만지작거린다. 그러다 외향형답게 친구를 불러내 **수육·갈비 맛집**으로 출동해 그동안 못 나눈 이야기를 나눈다.

## 🌙 포근한 저녁과 밤 (제안)
집에 돌아와 **피자**를 시켜놓고 게임 한 판, 혹은 보고 싶던 영화·드라마를 정주행한다. 맥주 한 캔과 함께 한 주의 피로를 리셋하고, 다음 주에 할 일만 슥 정리한 뒤 적당히 늦게 잠자리에 든다.

## ✨ 휴일을 보내는 나만의 규칙 3가지 (제안)
1. 평일 스트레스는 잊고, **하고 싶은 것만** 하기
2. 먹고 싶은 건 참지 않기 — **수육·갈비·피자 출동** 🍖
3. 취업 준비도 좋지만, **최소 반나절은 온전히 쉬기**

## 좌우명 (휴일)
- 유지 제안: “잘 쉬는 것도 실력이다” (현재 페이지 문구, 괜찮아서 유지)
- 대안(원하면): “쉴 땐 제대로, 할 땐 제대로” / 프로필 좌우명(“욕망은 정의다”)과 톤 맞추려면 이건 그대로 두는 게 무난.

## 페이지 반영 메모
- 3개 카드(아침/오후/저녁) 텍스트를 위 제안으로 교체.
- "하고 싶은 일" 체크리스트(TesseraJS localStorage)는 사용자가 직접 추가하는 인터랙션이라 seed 불필요(placeholder만 유지).
- 규칙 3개 + 좌우명 교체.

## 상태
- ✍️ 전부 **제안**이라 사용자가 한 번 훑고 OK/수정만 하면 확정. (위임받았지만 개인 루틴이라 확인 권장.)
